📲 모바일 PASMO/Suica 설정법 (부모님 폰에도 해드리기)

📱 iPhone (아이폰) 기준
  1. 이미 PASMO/Suica가 등록된 경우 → 그대로 사용 OK. 지갑(Wallet) 앱 → 잔액 확인 → 부족하면 Apple Pay로 충전.
  2. 신규 등록 (Suica 추천) — 지갑 앱 열기 → 우측 상단 + 버튼 → "교통카드" → "Suica" 선택 → 초기 충전 금액 입력 (1,000~10,000엔) → Apple Pay 결제로 충전 완료.
  3. 사용법 — 탑승구·운임통의 IC 마크에 폰 상단을 가볍게 대기. 얼굴 인식 해제도 불필요 (Express 모드 기본 On).
  4. 충전(차지) 방법 — 지갑 앱에서 Suica/PASMO 선택 → "충전" → 금액 선택 → Apple Pay 결제. 또는 편의점 "차지" 요청 (일본어: "차지 오네가이시마스").
🤖 Android 기준 Google Wallet에 모바일 Suica 또는 모바일 PASMO 등록 가능. 일본 발급 안드로이드폰 전용 앱도 있지만 한국 기종에선 Google Wallet 경유가 편함. 기종별 지원 여부는 설정 → Google Wallet 확인.
⚠️ 주의사항 ❗ JR 시코쿠 요산선(마쓰야마→다카마쓰)은 IC 불가 — 4/25 이동 구간이라 가장 주의. 마쓰야마역에서 직접 현금으로 티켓 구매하거나 온라인 사전 예약.
❗ 부모님 폰에 PASMO/Suica 있는지 확인 — 과거 일본 여행 시 등록했을 가능성. 지갑 앱 열어 확인.
❗ 잔액 부족 시 개찰 막힘 — 트램은 마지막에 잔액 부족하면 현금 추가 투입 필요. 최소 2,000엔 충전해두면 편함.
  • 4명 모두 폰에 PASMO/Suica/ICOCA 등록 확인 이미 있으면 잔액만 확인, 없으면 출국 전 또는 인천공항 도착 후 신규 등록.
  • 1인당 2,000~3,000엔 초기 충전 시코쿠에서 쓸 금액. 모자라면 편의점에서 추가 충전 가능.

🚃 이요테츠 노면전차 (마쓰야마 시내) 타는 법

  • 속옷·양말 4일분 + 여분 1일
  • 잠옷 / 실내복 호텔에 유카타 있을 수 있으나 개인 선호 시.
  • 세면도구 (칫솔·치약·면도기·세안제) 호텔 어메니티 확인 후 판단. 기내용 사이즈.
  • 자외선 차단제 4월도 자외선 강함. 나오시마 야외 활동 시 필수.
🔌 전자기기 & 통신
  • 휴대폰 & 충전기 (4인) 일본 콘센트는 한국과 동일(A타입). 별도 어댑터 불필요. 단 110V라 고출력 기기(드라이기 등)는 주의.
  • 포켓와이파이 또는 eSIM 공항 수령 시 수령 카운터 위치·예약 번호 확인.
  • 보조 배터리 (10,000mAh 이상) 나오시마 당일치기처럼 외출 긴 날 필수. 기내 반입만 가능, 위탁 불가!
  • 카메라 (선택) 나오시마 빨간·노란 호박, 지추 미술관은 촬영 금지지만 야외 작품은 OK.
  • 멀티 충전 케이블 + 멀티탭 가족용 멀티탭 1개면 호텔 콘센트 부족 해결.
💊 건강 & 상비약
  • 부모님 상시 복용약 혈압약·당뇨약 등 4일치 + 여분 2일치. 기내수하물에.
  • 상비약 세트 두통약·소화제·지사제·알레르기약·멀미약(페리용).
  • 밴드 / 반창고 나오시마 도보·자전거 — 발 물집 대비.
  • 파스·진통 패치 부모님 무릎·허리 부담 대비.
  • 마스크 기내 착용 + 일본 병원·공공시설 권장.
  • 손 소독제 / 물티슈
🛍️ 기타 & 편의
  • 여행용 캐리어 (4인) 제주항공 위탁수하물 규정 준수. 마쓰야마역 대형 코인로커(800엔/일) 사용 가능 사이즈.
  • 에코백 / 접이식 가방 일본은 비닐봉투 유료. 쇼핑·온천 후 젖은 빨래용.
  • 필기구 (볼펜) 기내 서류 작성용.
  • 지퍼백 (대·소 여러 장) 젖은 수건·속옷 분리용. 온천 방문 시 매우 유용.
  • 간식 / 음료 나오시마 섬 내 식당 적음. 초콜릿·에너지바 가볍게.

🧴 온천 갈 때 챙길 것 (4인 기준)
⚠️ 본관 카미노유(420엔)는 수건·샴푸 無 아스카노유(610엔, 한국 여권 무료)는 샴푸·린스·바디워시 구비. 수건은 어느 쪽이든 가지고 가는 게 안전.
  • 작은 수건(페이스 타월) 4장 몸 씻기·머리 위 올리기용. 호텔에서 한 장씩 챙기거나 100엔샵 구매.
  • 큰 수건(바디 타월) 4장 나와서 몸 닦기용. 호텔에 말하면 외출용 수건 대여 가능.
  • 100엔 동전 여러 개 아스카노유 신발장·쓰바키노유 락커(리턴식). 1인당 2~3개.
  • 갈아입을 속옷 (4인) 지퍼백에 담아 가방에.
  • 지퍼백·비닐봉지 여러 장 젖은 수건·속옷 담기.
  • 머리끈 긴 머리는 탕에 담그기 전 반드시 묶기 (매너).
  • 샴푸·비누 (선택) 본관 갈 계획이면. 아스카노유만 갈 거면 불필요.
  • 생수 / 음료 탕 후 수분 보충용.
